# StereoCamera
双目摄像头标定以及深度匹配

- **依赖库** ：*OpenCV3.0*(x64)：http://opencv.org/downloads.html

### 代码块
- StereoCamera：SGBM双目摄像头深度匹配
- CameraCalibration：双目摄像头标定程序
